== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. If you want best ticker in your wordpress theme. Place `<?php echo do_shortcode('yourshortcode');?>` in your template.

== For Basic usages==

<strong>General Ticker</strong>

<code>[ticker]</code>


<strong>1st Part Background Change</strong>

<code>[ticker id="2" color="blue" bg="green" speed="2000" ]</code>


<strong>2nd Part Background Change</strong>

<code>[ticker id="3" bg="red" color="black"speed="1500" ]</code>


<strong>Change Text</strong>

<code>[ticker id="4" text=" Best Ticker" color="green" bg="purple" speed="500" ]</code>


<strong>Post Count</strong>

<code>[ticker id="5" count="1" color="black" bg="fuchsia" speed="100" ]</code>


<strong>Ticker From Post Category</strong>

<code>[ticker id="6" category="classic"  text=" Top News"  color="chartreuse" bg="maroon" speed="1000" ]</code>


<strong>Cursor seed</strong>

<code>[ticker id="7" cursorspeed="2" color="maroon" bg="blue"]</code>


<strong>Ticker Speed Change</strong>

<code>[ticker id="8" speed="2000" color="red" bg="orange" text=" Test Ticker" ]</code>
